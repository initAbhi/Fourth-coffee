# Font Files Required

Please add the following font files to this directory:

## Agradable Sans
- `agradable-sans.woff2` (preferred format)
- OR `agradable-sans.woff` 
- OR `agradable-sans.ttf`

## Salina Trial
- `salina-trial.woff2` (preferred format)
- OR `salina-trial.woff`
- OR `salina-trial.ttf`

**Note:** The app will use system fonts as fallback until these files are added. The 404 errors in the console are expected and won't break the application.
